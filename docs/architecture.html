Makix Architecture

A hyper-deterministic, multi-phase agentic pipeline designed for zero-drift execution and 100% auditability.

The 7-Phase Protocol

01 CLI Invocation & Boot

Deterministic boot sequence with firewall validation and environment integrity checks.

02 Input Gatekeeper

Analysis of token density and trigger for scratchpad offloading to prevent context bloat.

03 Meta-Routing

Zero-temperature intent classification and dynamic skill manifest loading.

04 Planning

Pre-execution commitment to a JSON-structured roadmap with termination anchors.

05 Execution Loop

Parallel REACT reasoning loop with strict safety breaks (15 step limit).

06 Compression

Map-Reduce context compression to maintain technical precision while resetting token counters.

07 Memoria Ingestion

Durable knowledge extraction as SPO triplets and ingestion into the vector graph.

Skill Registry

Skills are self-contained logical units injected dynamically into the context. This modularity ensures the agent remains focused on the relevant tool schemas without overwhelming its memory.
