Full Settings Reference

Cowork provides granular control over reasoning, memory, and performance. All parameters below can be managed via cowork config or direct file editing.

Core Model & Logic

Setting Default Role
api_endpoint Target API gateway (e.g. Makix Edpt / OpenAI).
model_text Primary worker for reasoning (Qwen3-30B-A3B-2507).
model_router High-speed model for intent classification.
model_compress Model used for Map-Reduce compression.
embedding_model Used for Memoria Graph vector generation.

Runtime & Performance

Setting Value Description
max_steps 15 Safety limit for REACT loop turns.
max_total_tool_calls 30 Max aggregate tools per turn sequence.
max_tool_calls_per_step 5 Max parallel tool threads per step.
context_limit_tokens 6000 Threshold for context compression trigger.
request_delay_ms 0 Inter-turn delay to avoid rate limits.
retry_base_delay 2.0 Initial delay (s) for exponential backoff on retry.
max_retries 5 Total attempts for failed API calls.
max_concurrent_jobs 10 Limit for background cron/batch executions.

Inference Temperature

Setting Value Context
temperature_router 0.0 Must be deterministic for routing accuracy.
temperature_compress 0.1 Low creativity for factual summarization.
temperature_agent 0.4 Balanced for tool use and structured logic.
temperature_chat 0.7 Higher creativity for human interaction.

Memoria & Knowledge Graph

decay_rate 0.02 Temporal relevance decay factor.
top_k_memories 5 Count of memories injected into context.
memory_min_similarity 0.2 Similarity gate for RAG retrieval.
memory_min_weight 0.015 Min. weight for concept linkage.
memory_topic_overlap_min 1 Min overlap for cross-topic association.
memory_high_similarity_bypass 0.55 Threshold to ignore nearly identical inputs.
memory_kg_limit_triplets 100 Triple limit to prevent graph bloating.
memory_user_id UUID Unique identity for personalized memory.

Skills Runtime

skills_enabled Toggle the dynamic skill Registry (True/False).
skills_paths Custom directories for external .agent skills.
skills_router_min_score Min. confidence for activating a new skill (0.22).
skills_allow_category_expansion Allows agent to discover skills by category broad-match.
skills_max_metadata_skills Limit (64) for skills loaded in prime context.
skills_instruction_max_chars Upper limit for injected skill system prompt.
skills_max_resources_per_activation Limit for resource files read per tool call.
skills_resource_max_chars Limit for total resource content injected in prompt.

Miscellaneous

auto_save_important_refs: Persist scratchpad pointers to permanent storage.

idle_threshold_seconds: Time before session hibernation (900).

search_freshness: Web search time constraint (e.g. "1wk").

show_trace: Visual execution tracing for debugging (Boolean).

stream: Token-by-token UI response streaming (Boolean).

theme: Interface styling selection (dark/light).
