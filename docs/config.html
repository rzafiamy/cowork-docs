Configuration Reference

Cowork can be customized via the ~/.cowork/config.json file or environment variables. This section outlines the primary system settings.

Core Parameters

Parameter Default Description
model_text "gpt-4o-mini" The primary LLM used for reasoning and text generation.
max_steps 15 Safety limit for the number of REACT loops before termination.
context_limit 6000 The token threshold that triggers context compression.
temperature 0.4 LLM temperature for the main execution worker.

Environment Variables

OPENAI_API_KEY

Primary API key for model access. Overrides configuration file settings.

COWORK_DEBUG

Enables verbose system logs and detailed error traces when set to true.
