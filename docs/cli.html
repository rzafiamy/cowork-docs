CLI Reference

Cowork provides a comprehensive command-line interface for interaction, configuration, and auditing.

cowork chat

Interactive REPL

Starts an interactive agentic chat session where you can collaborate with the AI in real-time.

cowork chat

cowork run

One-Shot Task

Execute a specific task directly and exit. Perfect for automated workflows and quick fixes.

cowork run "Refactor utility.js to use ESM"

cowork ai

Profile Management

Manage AI profiles including endpoints, models, and API keys. Switch between different providers seamlessly.

cowork ai list
cowork ai switch gpt-4o

cowork sessions

History & Context

List, resume, or delete chat sessions. Sessions persist all state, including scratchpad and planning strategy.

cowork sessions list
cowork session resume [id]

cowork memory

Knowledge Graph

Inspect and query the persistent knowledge graph. See what the agent has learned from past interactions.

cowork memory list

cowork vector

RAG Embeddings

Manage vector database embeddings for Retrieval-Augmented Generation. 

cowork vector ingest path/to/docs

cowork firewall

Security Control

Audit and configure the tool execution firewall. Control which tools require handshake confirmation.

cowork firewall list
cowork firewall edit

cowork trace

Execution Audit

View detailed execution traces of the agent's internal reasoning and tool calls in a beautiful timeline.

cowork trace --latest

cowork cron

Scheduled Tasks

Schedule autonomous tasks to run periodically (daily, weekly, or once).

cowork cron add daily 09:00 "Generate daily summary"

cowork mm

Multi-Modal

Configure Vision, ASR (Speech-to-Text), and TTS (Text-to-Speech) service endpoints.

cowork mm status

cowork config

Settings

Manage global agent settings, API keys, and model preferences.

cowork config set model gpt-4o

cowork setup

Initialization

Run the interactive setup wizard to configure your agent environment. 

cowork setup

cowork tokens

Usage Tracking

Monitor cumulative token consumption across different models and sessions.

cowork tokens

cowork reset

Danger Zone

Wipe all local Cowork state and start fresh. Use with caution.

cowork reset
