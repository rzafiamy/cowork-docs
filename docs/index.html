Getting Started

Welcome to the Cowork documentation. Follow these instructions to set up your autonomous AI coworker on your local machine.

Prerequisites & System Requirements

Hardware

  • Minimal CPU & RAM (CLI is lightweight)
  • ~50MB disk storage (base installation)
  • External LLM API or Local LLM (Ollama)

Platform & Software

  • Linux, macOS, or Windows (WSL)
  • Python 3.10+
  • pip (Python package manager)

Installation

First, clone the repository and navigate to the CLI directory:

# Clone the repository
git clone https://github.com/rzafiamy/cowork.git
cd cowork/cli

Install the core package using pip. We recommend doing this inside a virtual environment:

pip install -e .

Optional Dependencies

Depending on the skills you want to enable, you can install optional extras (e.g. for local model embeddings):

  • pip install "cowork[documents]" # PDF, PPTX, XLSX, DOCX generation
  • pip install "cowork[local-rag]" # local vector search / embeddings
  • pip install "cowork[tools]" # Google APIs (Calendar, Drive, Gmail)
  • pip install "cowork[all]" # Install everything above

Configuration

Run the setup command to initialize your configuration folder (~/.cowork/) and set up your LLM provider:

cowork setup

Alternatively, you can export environment variables directly in your shell profile:

export OPENAI_API_KEY=sk-...
# For OpenAI compatible endpoints (Ollama, LM Studio etc)
export COWORK_API_ENDPOINT=https://api.openai.com/v1

Starting the Application

Once configured, you can start interacting with your autonomous coworker. The default interactive chat mode can be started by simply running:

cowork

Other useful starting commands:

  • cowork chat Explicitly start interactive chat.
  • cowork run "task" Run a one-shot task non-interactively.
  • cowork ping Test your LLM API connectivity to ensure everything is configured.
