Getting Started

Welcome to the Cowork documentation. Cowork is an autonomous AI coworker designed to execute complex tasks with a high degree of reliability and persistence.

Core Principles

Cowork is built on several key architectural pillars:

  • Deterministic Pipeline: A strict 7-phase execution model.
  • Persistent Memory: The Memoria knowledge graph.
  • Tool Augmented: Native skills for complex file operations and integrations.
